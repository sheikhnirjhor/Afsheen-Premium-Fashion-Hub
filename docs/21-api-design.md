# 21. API Design

### Endpoint: AI Chat Assistant

* **URL:** `/api/chat`
* **Method:** `POST`
* **Headers:** `Content-Type: application/json`

#### Request Payload
```json
{
  "prompt": "Suggest accessories for a navy blue suit."
}
{
  "reply": "A silver wristwatch and a crisp white silk pocket square would pair perfectly."
}
