# 11. Acceptance Criteria

* **AC-01 (AI Chat):** Chat responses must render within 3 seconds of user submission.
* **AC-02 (Cart Management):** Cart items must immediately update the total pricing mathematically.
* **AC-03 (Responsiveness):** UI layouts must adjust seamlessly from mobile viewport ($360\text{px}$) up to desktop ($1920\text{px}$).
