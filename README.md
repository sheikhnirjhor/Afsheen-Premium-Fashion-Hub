# Afsheen Premium Fashion Hub

## Project Overview

Afsheen Premium Fashion Hub is an e-commerce fashion platform designed for customers to browse premium fashion products, select sizes, check product availability, and place orders online.

## Features

* Product Browsing
* Product Search
* Size Selection
* Stock Availability Check
* Shopping Cart
* Online Payment (bKash, Bank Payment)
* Cash on Delivery
* Order Management

## Target Users

* Fashion Customers
* Business Owner
* Admin

## Technologies

* Frontend: HTML, CSS, JavaScript
* Backend: PHP / Laravel (or Node.js)
* Database: MySQL
