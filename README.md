# Afsheen Premium Fashion Hub

## Project Overviews

Afsheen Premium Fashion Hub is an e-commerce fashion platform designed for customers to browse premium fashion products, select sizes, check product availability, and place orders online.

## Features

* Product Browsing
* Product Search
* Size Selection
* Stock Availability Check
* Shopping Cart
* Booking Guideline
* Online Payment (bKash,Bank Payment,Nagad,Visa Card,Master Card,Debit Card)
* Voucher Coupon
* Cash on Delivery
* Order Management
* Return Policy

  
## Target Users

* Buyer
* Business Owner
* Moderator Panel
  

## Technologies

* Frontend: HTML, CSS, JavaScript
* Backend: PHP / Laravel (or Node.js)
* Database: FireBase
